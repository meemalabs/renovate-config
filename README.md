# Renovate Bot Meema Preset

Meema presets for the [Renovate](https://github.com/renovatebot/renovate) bot.

## Usage 💡

Add this into your `renovate.json` file:

```json
{
  "extends": [
    "@meemalabs"
  ]
}
```
